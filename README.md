# WeChat
实现类似微信朋友圈，评论回复，九宫格布局。处理键盘弹出后定位到当前点击的被评论人处。

1、缓存高度（感谢北京标哥的开源库－－>HYBMasonryAutoCellHeight），使页面滑动不卡，丝丝入滑。

2、页面全部采用Masonry自动布局。（自定义九宫格布局）
